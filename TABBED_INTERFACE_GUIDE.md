## Workflow

### Typical Usage Flow

1. **Start in Configure Tab** (default)
   - Enter your basic financial information
   - Add individual expenses in Expense Tracker
   - Optionally add life event transitions

2. **Automatic Switch to Results**
   - When you make changes, simulation runs automatically
   - App switches to Results tab to show updated projections
   - "Ready" badge appears on Results tab

3. **Review Results**
   - Check key metrics
   - Review charts and timeline
   - Look for warnings or issues

4. **Iterate**
   - Switch back to Configure tab
   - Adjust parameters
   - App automatically switches to Results again

### Manual Tab Switching

Click the tab headers at any time:
- **Configure** tab: Settings icon + "Configure" label
- **Results** tab: Chart icon + "Results" label + status badge

## Status Indicators

### Results Tab Badge
- **"Ready"** (green): Simulation has completed, results available
- **No badge**: No results yet, configure parameters first

### Empty States

**Configure Tab (no config)**:
- Welcome message
- Instructions to enter parameters
- Privacy notice

**Results Tab (no results)**:
- "No Results Yet" message
- Button to switch to Configure tab
- Instructions to configure parameters
